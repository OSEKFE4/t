# رفع الملفات

تطبيق ويب بسيط لرفع الملفات مع واجهة مستخدم سهلة الاستخدام.

## المميزات

- 📂 سحب وإفلات الملفات
- 📋 قائمة الملفات المختارة مع الأيقونات والأحجام
- ✅ شريط التقدم لكل ملف
- 🗑️ إمكانية إزالة الملفات قبل الرفع
- 📱 تصميم متجاوب يدعم الهواتف والأجهزة اللوحية

## الاستخدام

افتح ملف `index.html` في أي متصفح حديث.
